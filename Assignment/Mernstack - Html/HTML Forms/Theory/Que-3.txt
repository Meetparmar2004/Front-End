What is the purpose of the label element in a form, and how does it improve accessibility?

--> <label> : This element also help users who have difficulty clicking on very small regions (such as radio buttons or checkboxes) - 
              Because when the user clicks the text within the <label> element, it toggles the redio button or Checkbox.

--> Improve accessibility: 
        1. Making Forms Accessible :
            -> Clear input field guidance
            -> Accessible error message
            -> Improved form completion rates
            -> Assistive technology compatibility

        2. Improving Navigation and Interaction :
            -> Descriptive link text
            -> Accessible buttons and controls
            -> Semantic meaning

        3. Enhancing Usability for All Users :
            -> Reduced cognitive load
            -> Improved navigation
            -> Reduced error rates
